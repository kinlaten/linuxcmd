# File explorer
`:Explore` or `:E` + location

Open Explorer in split pane: `Vex` or `Hex`

Keymap:

| Key | Action |
| --- | --- |
| `Enter` | open file/dir |
| `-` | go up parent dir |
| `%` | create file |
| `d` | create directory |
| `D` | delete |
| `R` | rename |
| `mf` | mark file |
| `mc` | copy marked |
| `mm` | move marked |
| `q` | quit explorer |

To open a file in split pane instead current buffer: `v` (vertical), `o` (horizontal) or `t` (new tab)

To navigate between 
- pane: `Ctrl w + hjkl`
- tab: cmd `tabnext` / `tabn` or `tabp` or `tabn<number>`
